# 14848_HW4
Hadoop Map Reduce Temperature Dataset

- hw4_execution_1, hw4_execution_2, and hw4_execution_3 show hadoop's execution using our mapper and reducer on the temperature dataset. The images also show the creation of the OutputFolder and the merge command. 
- temperature_mapper.py is the mapper file.
- temperature_reducer.py is the reducer file.
- temperatureResults is the file resulting from merging all reducer output parts.
